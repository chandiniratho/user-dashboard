# User Dashboard

A responsive frontend dashboard that fetches and displays user data from a public API.

## Features

- Fetch users from API
- Display users in responsive cards
- Search users by name
- Sort users by name or company
- View user details on a separate page
- Loading and error handling
- Responsive design for mobile and desktop

## Technologies Used

- React
- Vite
- Axios
- React Router
- CSS

## API Used

https://jsonplaceholder.typicode.com/users

## Setup Instructions

Clone the repository

git clone https://github.com/chandiniratho/user-dashboard.git

Install dependencies

npm install

Run the project

npm run dev

## Deployment

The application is deployed on Vercel.
